This is the description of the openTCS API.

Some tutorials/code examples are available on the following pages:

  1. The class documentation for {@link org.opentcs.access.rmi.DynamicRemoteKernelProxy} explains how to create an RMI connection to communicate with a running openTCS kernel.
  2. For code examples explaining how to create and work with transport orders, see the {@link org.opentcs.data.order org.opentcs.data.order package documentation}.
  3. To learn how to create a vehicle driver, see the {@link org.opentcs.drivers org.opentcs.drivers package documentation}.
  4. To learn how to create a theme customizing the appearance of locations or vehicles in the plant overview client, see the {@link org.opentcs.util.gui org.opentcs.util.gui package documentation}.
